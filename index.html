Emergency Location Sensor

| C | Python | Linux | Raspberry Pi | SolidWorks |

Description:

1. A seismic sensor first detects ambient noise from the environment.
2. Multiple devices will be placed at the disaster site where the victim will begin knocking against the rubble.
3. Using digital signal processing (DSP), the desired signal is isolated from the noise.
4. Each device will then transfer this information to the host module using TCP communication.
5. This data is used in conjunction with our patented triagulation algorithm in order to locate the victim.

Reflow Soldering Oven

| C | Python | 8051 Assembly |

Description:

Designed & programmed a reflow oven controller in order to solder/fasten surface mount technology (SMT) components onto a PCB by following a specific temperature profile.
Sensors detected the temperature inside the reflow oven, which would then be regulated using pulse wave modulation (PWM).
Functionalities also included an LCD user interface (UI) that displayed temperature change, time elaspsed, and current status of the reflow process, as well as a fail stop.

Motorized Laser Controller

| C | MATLAB | Simulink | Altium | SolidWorks | Arduino |

Description:

Built and programmed PID controllers for self-made, mechanically commutated DC motors that were able to finely draw shapes using a laser.
Designed PCB and encoder wheel using Altium. Designed CAD parts using SolidWorks.
This encoder wheel was used to detect the position of the motor which consequently was used to program a honing mechanism and controller.
Modelled inverse kinematics in Simulink and analyzed a large amount of output data using Excel Spreadsheet in order to fine-tune accuracy.
